# Simple Calculator in Python

![preview.gif](.github/preview.gif)

For simplicity it supports only `+`, `*` operators and single digit numbers.

## Running
- `python ./calc.py 2+2*2`
- `chmod +x ./calc.py && ./calc.py 2+2*2`
